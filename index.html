⚙️ Configurar Sincronização em Nuvem
Como configurar (apenas 1x, pelo administrador):
1. Acesse jsonbin.io e crie uma conta gratuita
2. Vá em Account → API Keys → copie a Master Key
3. Clique em + Create Bin, cole {} e salve
4. Copie o BIN ID da URL (ex: 64a3bc...)
5. Distribua o arquivo HTML + BIN ID + API Key para sua equipe
6. Cada membro preenche este formulário no seu computador
🔒 As credenciais ficam salvas somente no seu navegador (localStorage). Não são enviadas a nenhum servidor além do jsonbin.io.
